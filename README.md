## 🏗️ How I Build

My approach to building AI agents diverges from standard reactive chatbots. I focus on **Deterministic Agentic Logic**:
- **Spec-Driven Development:** Agents act as execution engines driven by strictly defined Markdown/JSON specifications rather than open-ended prompts.
- **File-Based Persistent Memory:** Overcoming LLM context window collapse by utilizing the file system (and custom horizontal skills directories) as the agent's long-term memory and capability registry.
- **API & MCP Integration:** Agents interact directly with external environments (e.g., ClickUp, Slack, CRMs) via robust API callbacks and Model Context Protocol servers.
