### 🧠 My Core Architecture Philosophy
The standard Chat UI is broken due to context rot and lack of persistence. I architect systems using a **Directive-Orchestration-Execution** pattern:
1. **Directives:** Markdown SOPs that guide the reasoning.
2. **Orchestration:** Persistent, file-based memory structures (Rules, Contexts, Skills) that prevent context window collapse.
3. **Execution:** Deterministic Python tools that interact with APIs, perform complex math, and manipulate local files flawlessly.
